# Workmate AI チャットボットシステム

このプロジェクトは、企業向けAIチャットボットシステム「Workmate」です。

## プロジェクト構成

- `Chatbot-Backend-main/` - バックエンドAPI（FastAPI + Supabase）
- `Chatbot-Frontend-main/` - フロントエンドUI（React + Vite）
- `docs/` - ドキュメント類
- `scripts/` - 実行スクリプト類

## クイックスタート

### フロントエンド開発
```bash
npm run dev
```

### バックエンド開発
```bash
cd Chatbot-backend-main
python main.py
```
